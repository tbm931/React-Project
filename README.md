# מערכת ניהול פגישות לעסקים קטנים

פרויקט צד לקוח ב-React וצד שרת ב-Node.js, המיועד לעסקים קטנים. המערכת מאפשרת ניהול ושמירת נתונים בצורה פשוטה, תוך הטמעת לוגיקה עסקית שמפשטת את התהליך עבור בעלי העסקים.  
לקוחות יכולים לחפש בקלות שירותים מתאימים ולזמן פגישות ישירות מול העסק, בעוד שמנהלי המערכת יכולים לצפות ולנהל את כל הפגישות שנקבעו.

להפעלת הפרויקט יש להתקין את התלויות הבאות:  
- Sequelize:  
  ```bash
  npm install sequelize
ESLint (ליצירת קונפיגורציה):


npm init @eslint/config@latest
Jest (לבדיקות יחידה):

npm install --save-dev jest
השימוש העיקרי במערכת הוא לקביעת פגישות עבור שירותים המוצעים על ידי עסקים שונים, וכן לאפשר למנהלים לצפות ולנהל את כל הפגישות שנקבעו.

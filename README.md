**פרויקט ב React (צד לקוח) ו Node (צד שרת)**
הפרויקט מאפשר שמירת נתונים והתעסקות עם לוגיקה עבור עסקים קטנים - 
כך שלא נדרשים לעשות זאת בעצמם,
ולקוחות יכולים למצוא בקלות את השירותים שהם צריכים ולקבוע אליהם פגישות.


**התקנות הנדרשות:**
SEQULIZE:  npm install sequelize
ESLINT:    npm init @eslint/config@latest
JEST:      npm install --save-dev jest

**שימוש:**
קביעת פגישה עבור שירותים מעסקים שונים.
צפיה בכל הפגישות המאופשרת למנהלים.
